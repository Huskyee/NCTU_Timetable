## NCTU Timetable

交大選課預排系統

### 功能

1. 搜尋課名 / 課號 / 老師
2. 點選「搜尋結果」或「已選課程」或「時間表」中的課程名字可以查看詳細資訊
3. 將滑鼠移動至「搜尋結果」的課程上會顯示是否衝堂
4. 點選「+」或「-」將課程加入課表或從課表移除。
5. 將課表存成圖片
6. 關閉瀏覽器後下次再打開課表不會消失
